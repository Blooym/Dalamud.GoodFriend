GoodFriend

Allows for in-game friends to communicate login and logout events between eachother safely.

Using GoodFriend

  1. Make sure you have Dalamud setup correctly.
  2. Load your game and navigate to the Dalamud Plugin Installer.
  3. Search for the plugin "GoodFriend" in the list of plugins.
  4. Download the plugin.
  5. Done! Please note that GoodFriend only works with friends that are also using the plugin as well.

Privacy

GoodFriend does not store any identifiable information, period. Anything remotely identifable (such as CIDs) are hashed by before being transmitted. If any information is not correctly hashed by the sender it will not be accepted or relayed.
